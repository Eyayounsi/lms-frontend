FAQ

Questions fréquentes

Tout ce que vous devez savoir sur notre plateforme.

Comment m'inscrire à un cours ?

Créez un compte gratuit, parcourez notre catalogue, et cliquez sur « S'inscrire ». Vous pouvez commencer immédiatement.

Combien de temps ai-je accès à un cours ?

Une fois inscrit, vous avez un accès illimité au cours. Revenez quand vous le souhaitez.

Quels moyens de paiement acceptez-vous ?

Nous acceptons les cartes bancaires (Visa, Mastercard) via Stripe. Paiement sécurisé et instantané.

Vais-je obtenir un certificat ?

Oui ! À la fin de chaque formation, vous recevez un certificat avec un code de vérification unique.

Comment vérifier un certificat ?

Rendez-vous sur « Vérifier Certificat » dans le menu, entrez le code unique et consultez les informations validées.
